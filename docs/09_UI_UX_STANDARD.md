# 📂 Archie-G: UI/UX 디자인 및 컴포넌트 표준 (UI/UX Standard)

Archie-G의 프리미엄 다크 모드 인터페이스와 사용자 경험을 유지하기 위한 지침입니다. 모든 개발자는 UI 수정 및 신규 컴포넌트 추가 시 본 표준을 준수해야 합니다.

---

### 1. 디자인 미학 (Aesthetics)
Archie-G는 전문적인 아키텍처 분석 도구로서 고도의 정밀함과 현대적인 미감을 동시에 전달해야 합니다.

*   **테마 (Theme)**: Deep Dark Mode를 기본으로 하며, 순수한 블랙보다는 미세한 네이비 톤이 섞인 다크 그레이를 사용합니다. (배경: `#050508`, `#0A0A10`)
*   **글래스모피즘 (Glassmorphism)**: 
    *   모든 패널은 `backdrop-blur-xl` 속성을 가져야 합니다.
    *   배경색은 `white/[0.03]` 정도의 낮은 투명도를 유지하여 하위 레이어가 은은하게 투영되도록 합니다.
*   **보더 (Border)**: 
    *   강한 선보다는 빛에 반사된 듯한 얇은 실선(`white/[0.06]`)을 사용합니다.
    *   중요 요소 강조 시에는 그라데이션 보더를 활용합니다.
*   **색상 팔레트 (Palette)**:
    *   **Violet (Primary)**: 창의성과 지능을 상징. 히스토리, 주요 액션 버튼에 사용.
    *   **Blue (Secondary)**: 신뢰와 기술을 상징. 분석 프로세스, 상태 표시기 등에 사용.
    *   **Yellow (Highlight)**: 마스터 플랜, 경고, 핵심 요약 등에 사용.

### 2. 컴포넌트 규격 (Component Specs)
*   **탭 시스템 (Tabs)**:
    *   현재 활성화된 탭은 배경색과 폰트 색상을 반전시켜 명확히 구분합니다.
    *   전환 시 `Framer Motion`의 `AnimatePresence`를 사용하여 좌우 슬라이드 또는 페이드 효과를 적용합니다.
*   **카드 (Cards)**:
    *   `rounded-3xl` 이상의 큰 곡률을 사용하여 부드러운 인상을 줍니다.
    *   마우스 호버(Hover) 시 `scale-105` 또는 보더의 발광 효과를 추가하여 인터랙티브함을 강조합니다.
*   **타이포그래피 (Typography)**:
    *   본문: `Inter` 또는 `Pretendard`와 같은 가독성 높은 산세리프 폰트.
    *   코드 및 데이터: `JetBrains Mono` 또는 `Roboto Mono`를 사용하여 전문성을 부여합니다.

### 3. 사용자 경험 원칙 (UX Principles)
*   **실시간성 (Real-time)**: 분석 중에는 로딩 바 대신 실제 데이터가 스트리밍되는 모습을 보여줌으로써 시스템이 활발히 작동 중임을 인지시킵니다.
*   **영속성 (Persistence)**: 사용자가 공들여 분석한 결과는 언제든 다시 찾아볼 수 있도록 '저장(PIN)' 기능을 항상 노출합니다.
*   **공간 효율성 (Space)**: 복잡한 정보를 다루는 만큼, 접기/펴기 기능이나 탭 전환을 통해 화면 공간을 효율적으로 사용합니다.

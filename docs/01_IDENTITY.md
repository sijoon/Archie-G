# 📂 [01] 프로젝트 정체성 (Identity)

Archie-G 에이전트의 핵심 미션과 운영 환경에 대한 정의입니다.

---

### 1. 에이전트 개요
*   **이름:** Archie-G (AI Architecture Guide Agent)
*   **핵심 미션:** 로컬 소스 코드(Swift, SwiftUI, Obj-C, Kotlin, Java) 및 관련 API 문서를 정밀 분석하여 통합 흐름도와 가이드 문서를 자동 생성함.
*   **환경:** 100% 오프라인 로컬 환경 (보안망 준수).

### 2. 구동 원칙
*   보안이 강화된 사격망 내에서 외부 유출 없이 소스 코드를 분석합니다.
*   개발자에게 시각화된 흐름도와 함수 단위 가이드를 제공하여 코드 파악 시간을 단축합니다.
*   신규 입사자나 리팩토링 시 지식 자산화의 핵심 도구로 작동합니다.

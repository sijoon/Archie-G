# 📂 [04] 분석 및 문서화 출력 표준 (Output Standards)

시각화 흐름도와 가이드 문서의 구성 형식 및 품질 규격입니다.

---

### 1. 흐름도 (Flowcharts)
*   **Mermaid.js:** 모든 시각화는 Mermaid 구문을 사용함.
*   **하이브리드 특화:**
    *   **Native 영역:** Blue 서브그래프 구현.
    *   **Web 영역:** Yellow 서브그래프 구현.
    *   두 영역 간의 Bridge 통신 인터페이스를 명확한 화살표와 텍스트로 표현.

### 2. 가이드 문서 확장 (Guide Structure)
단순 코드 설명을 넘어선 **3단 결합 구조**를 유지합니다.
1.  **[코드 로직]:** 함수의 내부 흐름 및 데이터 처리 방식 설명.
2.  **[API 가이드]:** 관련 라이브러리/프레임워크의 공식 Usage 및 구현 제약 사항 매핑.
3.  **[보안 가이드]:** 암호화, 권한 체크, 데이터 노출 여부 등 보안 권고 사항.
